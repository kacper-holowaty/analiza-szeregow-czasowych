# Analiza szeregów czasowych

## [Laboratorium 1](lab01/) 
Analiza dekompozycji szeregu czasowego wartości dodanej brutto (Gross Value Added - GVA) dla Grecji. Obejmuje usunięcie trendu i sezonowości z danych kwartalnych oraz identyfikację składnika losowego.

## [Laboratorium 2](lab02/)
Analiza procesu rebazowania (zmiany roku bazowego) wartości dodanej brutto (Gross Value Added - GVA) dla Grecji. Celem jest przeliczenie danych z cen stałych roku 2020 na ceny stałe roku 2015 i 2024, aby umożliwić porównania z innymi danymi wykorzystującymi te lata bazowe.

## [Laboratorium 3](lab03/)
Analiza szeregu czasowego wartości dodanej brutto (Gross Value Added - GVA) dla Grecji wyrażonej w cenach stałych poprzez porównanie trzech modeli trendu - liniowego, potęgowego i wykładniczego.
